# homebrew-vcf

Homebrew tap for the [Vibe Coding Framework MCP](https://github.com/Kaelith-Labs/vcf-cli) CLI (`@kaelith-labs/cli`).

## Install

```bash
brew tap kaelith-labs/vcf
brew install vcf-cli
```

The formula installs the current alpha release. Pre-1.0 means breaking changes are possible between minor bumps — pin a specific version if you need reproducibility.

## Uninstall

```bash
brew uninstall vcf-cli
brew untap kaelith-labs/vcf
```
